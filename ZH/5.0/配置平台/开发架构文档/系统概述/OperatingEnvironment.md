# 运营环境
## 软件环境

| 软件名称    | 版本   | 备注 |
|------------|--------|---------------|
| Linux      | 2.6+   | 操作系统,指明内核版本不低于2.6，发行版(Ubuntu or Centos or suse)并不重要。 |
| Zookeeper  | 3.4.11 | 服务发现、配置发现 |
| Redis      | 3.2.11 | 配置平台缓存、session 保存 |
| MongoDB    | 2.8.0  | 配置平台数据库 |
| Redis      | 1.4.4  | 配置平台数据、会话存储 |
| Supervisor | 3.3.1  | 进程托管 |
