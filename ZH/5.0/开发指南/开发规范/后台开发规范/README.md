2. 在以下章节中，将用 `【必须】` 标签来标记开发过程中必须遵守的规范，其他均为建议

# 关于 Python 版本

**2020年以后，Python2 将不会有官方的支持和修复**

- **新项目** - 必须使用 Python 3 进行开发（推荐版本: Python 3.6）
- **老项目(仍在迭代更新)** - 需要尽快迁移至 Python 3。迁移方案请参考[Python 2to3 迁移指南](../最佳实践/2to3/)
- **老项目(处于稳定维护状态)** - 可维持当前 Python 版本
