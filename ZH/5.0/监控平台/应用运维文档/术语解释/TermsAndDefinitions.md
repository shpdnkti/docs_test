# 术语和定义

| 术语           | 解释                                               |
| -------------- | ------------------------------------------------- |
| monitor        | 监控告警服务后台                                    |
| transfor       | 数据清洗模块                                        |
| influxdb-proxy | influxdb 代理                                      |
| ESB            | 蓝鲸 PAAS 企业总线，异构系统间的 API 调用需要通过 ESB |
