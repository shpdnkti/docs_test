# 告警数据统计

URL: http://127.0.0.1:8081/fta/status/archive/

请求方法：GET

请求参数：无

返回字段说明：

| **字段**      | **含义**     |
|---------------|--------------|
| alarm_type    | 告警类型     |
| date          | 归总时间     |
| failure_type  | 失败类型     |
| is_success    | 是否成功处理 |
| solution_type | 套餐类型     |
| source_type   | 告警源       |
| sub_consumed  | 归总耗时     |
| sub_count     | 归总数       |
| sub_profit    | 归总收益     |

示例:
```bash
$ curl http://127.0.0.1:8081/fta/status/archive/

[
    {
        "alarm_type": "api_default",
        "date": "2017-05-26 00:00",
        "failure_type": "",
        "is_success": 1,
        "solution_type": "diy",
        "source_type": "REST-API",
        "sub_consumed": 293,
        "sub_count": 12,
        "sub_profit": 10507
    }
]
```
