# Tspider
Tspider 节点底层为基于 MySQL 的分布式关系型数据库存储。

图例，Tspider 节点

![](../../../../assets/dataflow/components/storage/dataflow-tspider.png)

#### 使用方式
- 节点名称： 自动生成，由上游结果表和当前节点类型组成
- 结果数据表：从上游节点继承过来
- 存储集群：通常可选有默认集群组集群，其它可选集群与任务所属项目相关
- 过期时间：数据入库后保存的过期时间

除此之外，Tspider 节点可根据上游节点结果表的输出情况，指定字段是否为索引字段。
