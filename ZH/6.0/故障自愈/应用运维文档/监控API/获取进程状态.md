# 获取进程状态(supervisord)

URL: http://127.0.0.1:8081/fta/status/process/

请求方法：GET

请求参数：无

返回格式：

`{process1_name: process1_status, process2_name: process2_status, }`


示例:

curl http://127.0.0.1:8081/fta/status/process/
